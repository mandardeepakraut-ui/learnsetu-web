# Design System: LearnSetu AI & Data Science Platform (Light Editorial White Edition)

## 1. Visual Theme & Atmosphere
A clean, light editorial interface with crisp typography, generous whitespace, asymmetric split layouts, and hardware-accelerated 3D WebGL particle motion. The atmosphere is warm, clinical, and gallery-airy — like a modern Scandinavian architecture studio.

- **Density:** 4 (Art Gallery Airy & Space-Conscious)
- **Variance:** 8 (Offset Asymmetric Grid & Interactive 3D Spline/Canvas Elements)
- **Motion:** 8 (Perpetual 3D Neural Nodes, 3D Tilt Cards, Interactive Career Hikes)

## 2. Color Palette & Roles
- **Canvas Crisp White** (`#FAFAFC` / `#FFFFFF`) — Primary background surface
- **Pure Surface Fill** (`#FFFFFF`) — Card containers, elevated panels, and modal backdrops
- **Deep Ink Primary Text** (`#0F172A`) — Primary headlines & track-tight display headers
- **Muted Slate Text** (`#64748B`) — Subtitles, curriculum topics, and metadata
- **Soft Whispered Border** (`rgba(15, 23, 42, 0.08)`) — 1px container dividers
- **Royal Electric Blue Accent** (`#0067FF`) — Primary CTAs, active highlights, focus rings
- **Indigo Accent Highlight** (`#4F46E5`) — 3D mesh nodes, status indicators

*Mandatory Rules:* Saturation < 80%. No dark mode canvas, no purple neon glows.

## 3. Typography Rules
- **Display / Headlines:** `Outfit` / `Satoshi` — Track-tight (-0.03em), weight-driven hierarchy.
- **Body Text:** `Plus Jakarta Sans` / `Geist` — Relaxed leading (1.6), max 65ch width for high readability.
- **Code & Metrics:** `JetBrains Mono` — Used for course duration (24 Weeks), course fee (₹14,999), and salary metrics.

## 4. 3D Elements & Interactive Features
- **3D Hero Neural Mesh:** Interactive WebGL/Canvas neural node network floating with mouse spring rotation.
- **3D Tech Stack Sphere:** Drag-to-rotate 3D tech universe (Python, ML, SQL, Power BI, PyTorch, LLMs).
- **3D Interactive Career Hikes & EMI Estimator:** Live salary hike calculator + ₹14,999 fee EMI breakdown with 3D tilt cards.
- **Interactive Course Matcher Quiz:** Instant 3-step recommendation filter.
- **3D Certificate Preview:** Live student name preview on 3D perspective certificate.
